You are running a Hadoop MapReduce program to calculate the total login time of each user.

Example:

user1,120
user1,30
Means:

user1 logged in for 120 minutes

then again for 30 minutes

Total = 150

Hadoop will automatically calculate totals for all users.

Mapper Class
public static class TokenizerMapper
        extends Mapper<Object, Text, Text, IntWritable>
Meaning
Mapper reads data line by line.

Input:

user1,120
Output:

(user1,120)
Variables
private Text user = new Text();
private IntWritable time = new IntWritable();
Creates Hadoop variables.

map() Function
public void map(Object key, Text value, Context context)
This function runs for every line.

Convert line into string
String line = value.toString();
Example:

"user1,120"
Split using comma
String[] parts = line.split(",");
Now:

parts[0] = user1
parts[1] = 120
Check correct format
if(parts.length == 2)
Ensures line has exactly 2 values.

Store username
user.set(parts[0]);
Stores:

user1
Convert time into integer
int loginTime = Integer.parseInt(parts[1]);
Converts:

"120"
into:

120
Store login time
time.set(loginTime);
Send output
context.write(user, time);
Mapper output:

(user1,120)
(user2,50)
(user1,30)
Reducer Class
public static class IntSumReducer
Reducer combines values of same key.

Input to reducer:

user1 → [120,30]
user2 → [50,100]
reduce() Function
public void reduce(Text key,
                   Iterable<IntWritable> values,
                   Context context)
Runs once per user.

Sum Variable
int sum = 0;
Stores total login time.

Loop Through Values
for(IntWritable val : values)
Reads all values.

Example:

120
30
Add Values
sum += val.get();
Calculation:

0 + 120 = 120
120 + 30 = 150
Store Result
result.set(sum);
Write Final Output
context.write(key, result);
Final output:

user1 150
user2 150
user3 220
user4 250
Main Method
public static void main(String[] args)
Starts the Hadoop job.

Create Configuration
Configuration conf = new Configuration();
Loads Hadoop settings.

Create Job
Job job = Job.getInstance(conf, "Login Time Analysis");
Creates job named:

Login Time Analysis
Set Main Class
job.setJarByClass(LoginTimeAnalysis.class);
Tells Hadoop which class to run.

Set Mapper
job.setMapperClass(TokenizerMapper.class);
Uses your Mapper.

Set Combiner
job.setCombinerClass(IntSumReducer.class);
Mini reducer.

Improves speed by reducing data before sending to reducer.

Set Reducer
job.setReducerClass(IntSumReducer.class);
Uses Reducer.

Output Types
job.setOutputKeyClass(Text.class);
job.setOutputValueClass(IntWritable.class);
Final output types:

Key → Text

Value → IntWritable

Input Path
FileInputFormat.addInputPath(job, new Path(args[0]));
Reads input from:

/input
Output Path
FileOutputFormat.setOutputPath(job, new Path(args[1]));
Writes result to:

/output
Run Job
System.exit(job.waitForCompletion(true) ? 0 : 1);
Runs job.

Success → 0

Failure → 1

1. What is Hadoop?
Apache Hadoop is an open-source framework used to store and process very large amounts of data across multiple computers.

Features
Handles Big Data

Distributed storage

Parallel processing

Fault tolerant

Scalable

Example
Instead of one computer processing 1 TB data, Hadoop divides work among many computers.

2. What is HDFS?
Hadoop Distributed File System is the storage system of Hadoop.

Work
Stores large files

Splits files into blocks

Saves blocks on multiple machines

Main Components
NameNode → Manages metadata

DataNode → Stores actual data

Advantage
If one machine fails, data is still safe because copies are stored elsewhere.

3. What is YARN?
Apache Hadoop YARN manages resources in Hadoop.

Work
Manages CPU and memory

Schedules jobs

Monitors tasks

Components
ResourceManager

NodeManager

Simple Meaning
YARN is like the “manager” of Hadoop cluster resources.

4. What is Mapper?
Mapper is a class in MapReduce that processes input data line by line and converts it into key-value pairs.

Example
Input:

user1,120
Mapper Output:

(user1,120)
Role
Reads data

Processes data

Generates intermediate output

5. What is Reducer?
Reducer combines and processes values having the same key.

Example
Input:

user1 → [120,30]
Reducer Output:

user1 150
Work
Receives grouped data

Performs calculations

Produces final result

6. What is Combiner?
Combiner is a mini reducer that runs after mapper.

Purpose
Reduces amount of data sent to reducer.

Advantage
Faster processing

Less network traffic

Example
Before Combiner:

(user1,120)
(user1,30)
After Combiner:

(user1,150)
7. Why do we create JAR files?
JAR (Java Archive) file packages all .class files into one file.

Purpose
Easy to run program

Hadoop executes JAR files

Example
jar -cvf logintime.jar *.class
Creates:

logintime.jar
8. Why delete output folder before running?
Hadoop does not overwrite existing output folders.

If /output already exists, job fails.

So we remove it using:

hdfs dfs -rm -r /output
9. What is Shuffle and Sort?
Shuffle and Sort is the phase between Mapper and Reducer.

Work
Groups same keys together

Sorts intermediate data

Example
Mapper Output:

(user1,120)
(user2,50)
(user1,30)
After Shuffle:

user1 → [120,30]
user2 → [50]
10. Difference between HDFS and Local Storage?
HDFS	Local Storage
Distributed	Single machine
Stores huge data	Limited storage
Fault tolerant	Less fault tolerance
Used in Hadoop	Used in normal systems
Faster for Big Data	Slower for Big Data
11. What are Key-Value Pairs?
Data format used in MapReduce.

Structure
(key,value)
Example
(user1,120)
Where:

Key = user1

Value = 120

12. Why use IntWritable instead of int?
Hadoop uses special writable classes for serialization.

Example
Java Type	Hadoop Type
int	IntWritable
String	Text
Reason
Hadoop can efficiently transfer data across network using writable classes.

13. What is the role of Configuration class?
Configuration conf = new Configuration();
Work
Loads Hadoop configuration settings.

Example
HDFS settings

YARN settings

Cluster information

14. Why use context.write()?
Used to send output from Mapper or Reducer.

Syntax
context.write(key,value);
Mapper Example
context.write(user,time);
Reducer Example
context.write(key,result);
15. What happens between Mapper and Reducer?
Between Mapper and Reducer, Hadoop performs:

Shuffle

Sort

Grouping

Example
Mapper Output:

(user1,120)
(user1,30)
After Grouping:

user1 → [120,30]
Reducer then processes grouped data.
